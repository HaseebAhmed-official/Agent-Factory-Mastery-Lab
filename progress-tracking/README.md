# Progress Tracking

> **Purpose**: Study metrics, analytics, and performance tracking
> **Generated by**: Status command and analytics dashboard
> **Format**: Markdown reports and JSON data

---

## What's Tracked?

This directory contains comprehensive learning analytics:

### 1. Progress Metrics
- Lessons completed (count, percentage)
- Chapters completed
- Current lesson and checkpoint
- Overall course completion

### 2. Study Habits
- Session frequency (days active per week)
- Session duration (average time per session)
- Study consistency (streak tracking)
- Peak productivity times

### 3. Checkpoint Activity
- Total checkpoints created
- Average checkpoints per lesson
- Depth coverage (% lessons with L3)
- Checkpoint quality scores

### 4. Quiz Performance
- Quizzes taken (count)
- Average score across all quizzes
- Improvement trend (first vs recent)
- Weak areas identified
- Topics mastered

### 5. Content Creation
- Total words written (revision notes)
- Files created (notes, flashcards, presentations)
- Quick references generated
- Flashcard decks created

### 6. Time Investment
- Estimated study time (word count / reading speed)
- Time per lesson (estimated)
- Total course hours invested
- Projected completion timeline

## Status Reports

**Generated by**: `Status` command

**Format**: Markdown snapshot of current state

**Frequency**: On-demand (user runs `Status`)

**Content**:
1. Overall progress (percentage complete)
2. Chapter breakdown
3. Completed lessons list
4. Current lesson status
5. Learning metrics
6. Mastery level assessment
7. Areas for improvement
8. Recommended next steps
9. Study habits analysis
10. Achievements unlocked

**Example**:
```markdown
# Progress Report: 2026-03-01

## Summary
- **Course Completion**: 12% (5/42 lessons)
- **Current Lesson**: 3.23 - CoWork Getting Started (L1 complete)
- **Average Quiz Score**: 82% (B+)
- **Study Time**: ~13.5 hours

## Chapter Progress
- Chapter 1: 0/6 (0%) - Skipped
- Chapter 2: 0/8 (0%) - Skipped
- Chapter 3: 5/12 (42%) - IN PROGRESS
- Chapter 4-6: Locked

## Recommendations
1. Take quizzes for 3.17, 3.22, 3.23
2. Push 3.17 and 3.22 to L3 depth
3. Complete Chapter 3 (7 more lessons)
```

## Achievements System

**File**: `achievements.json`

Gamified milestone tracking:

**Badge Categories**:
1. **Completion Badges**
   - First Lesson Complete
   - Chapter Champion (complete chapter)
   - Course Completer (all 42 lessons)

2. **Quality Badges**
   - Quiz Ace (90%+ average)
   - Checkpoint Master (10+ checkpoints)
   - Depth Explorer (all lessons to L3)

3. **Consistency Badges**
   - 7-Day Streak
   - 30-Day Streak
   - 100-Day Streak

4. **Improvement Badges**
   - Quiz Improver (+10% improvement)
   - Fast Learner (complete lesson in <3 hours)
   - Thorough Scholar (all quizzes + exercises complete)

## Using Analytics

### View Progress
```bash
# Generate status report
User: Status

# View analytics dashboard (HTML)
python scripts/analytics-dashboard.py --export-html
open progress-tracking/dashboard.html
```

### Track Improvement
1. Run `Status` weekly
2. Compare snapshots over time
3. Identify trends (improving vs plateauing)
4. Adjust study habits accordingly

### Identify Weak Areas
1. Check quiz performance section
2. Filter by topic or concept
3. Review lessons for weak topics
4. Reattempt quizzes

### Stay Motivated
- Unlock achievements
- Watch completion percentage grow
- Celebrate streak milestones
- Compare week-over-week progress

## Maintenance

### Update Frequency
- **On-demand**: User runs `Status` command
- **Automatic**: After `Finish` command (updates analytics)
- **Manual**: Run analytics script directly

### Cleanup
```bash
# Archive old status reports
mkdir progress-tracking/archive
mv progress-tracking/status-report-2026-*.md progress-tracking/archive/

# Reset analytics (start fresh)
rm progress-tracking/analytics-data.json
# Will regenerate on next `Status`
```
