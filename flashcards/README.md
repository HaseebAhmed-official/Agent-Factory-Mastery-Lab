# Flashcards for Spaced Repetition

> **Purpose**: Anki-compatible flashcard decks for long-term retention
> **Generated by**: Finish command (Stage 6 in finish-synthesis.md)
> **Format**: JSON (Anki-compatible)

---

## What Are Flashcards?

Flashcards use **spaced repetition** to help you remember concepts long-term:

- **Vocabulary cards**: Term → Definition
- **Concept cards**: Question about concept → Full explanation
- **Anti-pattern cards**: What goes wrong? → Problem + Fix
- **Framework cards**: Explain framework → Formula + Application

---

## File Naming Convention

`lesson-{X.Y}-deck.json`

**Examples**:
- `lesson-3.1-deck.json` (Lesson 3.1: Claude Code Origin Story)
- `lesson-3.17-deck.json` (Lesson 3.17: Ralph Wiggum Loop)

## How to Use

### Option 1: Review Online (JSON)
Read flashcards directly from JSON file:
- Open `lesson-3.1-deck.json`
- Review questions and answers
- Self-assess your recall

### Option 2: Import to Anki (Recommended)

**Anki** is a free, open-source spaced repetition app:

1. **Install Anki**: [ankiweb.net](https://apps.ankiweb.net/)
2. **Convert JSON to Anki format**:
   ```bash
   python scripts/export-flashcards-to-anki.py flashcards/lesson-3.1-deck.json
   ```
   This creates `flashcards/_anki-export/lesson-3.1.apkg`

3. **Import to Anki**:
   - Open Anki
   - File → Import
   - Select `lesson-3.1.apkg`

4. **Review with spaced repetition**:
   - Anki automatically schedules cards
   - Review daily (5-15 min)
   - Long-term retention improves dramatically

## Card Types

### Vocabulary Cards
**Front**: "What is {term} in the context of {domain}?"
**Back**: "{Definition}"
**Difficulty**: Easy

### Concept Cards
**Front**: "Describe {concept name} and its purpose."
**Back**: "{What it is + Why it matters + How it works}"
**Difficulty**: Medium

### Anti-Pattern Cards
**Front**: "What goes wrong if {specific misuse}?"
**Back**: "{Problem}<br><strong>Fix:</strong> {Correct approach}"
**Difficulty**: Hard

### Framework Cards
**Front**: "Explain the {framework name} framework."
**Back**: "{Formula/Pattern}<br>{Application guidance}"
**Difficulty**: Medium

## Generation Process

Flashcards are **automatically generated** when you use the `Finish` command:

1. System reads ALL checkpoint part files for the lesson
2. Extracts vocabulary terms, key concepts, anti-patterns, frameworks
3. Generates flashcard pairs (front/back) for each
4. Tags cards by lesson, layer, topic, difficulty
5. Saves Anki-compatible JSON to this directory

You do NOT need to manually create flashcards.

---

## Study Workflow

**Recommended schedule**:

1. **Day 0** (Lesson complete): Generate flashcards with Finish command
2. **Day 1**: First review (all cards)
3. **Day 3**: Second review (cards you struggled with)
4. **Day 7**: Third review (spaced repetition)
5. **Day 14**: Fourth review
6. **Day 30**: Fifth review
7. **Ongoing**: Anki schedules cards automatically based on your performance

## Export Scripts

### Convert JSON to Anki (.apkg)

**Script**: `scripts/export-flashcards-to-anki.py`

**Usage**:
```bash
python scripts/export-flashcards-to-anki.py flashcards/lesson-3.1-deck.json
```

**Output**: `flashcards/_anki-export/lesson-3.1.apkg`

### Merge Multiple Lessons

**Combine multiple lesson decks into one**:
```bash
python scripts/merge-flashcard-decks.py \
  flashcards/lesson-3.1-deck.json \
  flashcards/lesson-3.2-deck.json \
  flashcards/lesson-3.3-deck.json \
  --output flashcards/chapter-3-complete.json
```

## Statistics

Each deck JSON includes metadata:
- **Total cards**: Count of flashcards
- **Layers covered**: L1, L2, L3...
- **Difficulty distribution**: Easy/Medium/Hard breakdown
- **Card types**: Vocab/Concept/Anti-pattern/Framework counts

---

## Maintenance

Flashcards are regenerated each time you use the Finish command for a lesson.

If you update lesson notes and want to refresh flashcards:
1. Use `Finish` command again (will overwrite existing deck)
2. OR manually edit the JSON file
3. Re-import to Anki to update cards
