# Named Anti-Patterns

> **Cross-refs**: [Edge Case Protocol](edge-case-protocol.md) | [Cross-Chapter Map](cross-chapter-map.md)

Reference these by name whenever relevant throughout the curriculum:

| Anti-Pattern | Description | Source |
|-------------|-------------|--------|
| **Premature Specialization** | Building Custom Agents before requirements are stable through incubation | Ch. 1 |
| **Perpetual Incubation** | Using General Agents for production workloads indefinitely, never graduating to Custom | Ch. 1 |
| **Skipping Incubation** | Attempting to write specifications without exploration and discovery | Ch. 1 |
| **Vibe Coding** | Writing code through ad-hoc AI prompting without specifications | Ch. 5 |
| **Context Stuffing** | Overloading CLAUDE.md with irrelevant information, exceeding signal-to-noise threshold | Ch. 4 |
| **Context Starvation** | Providing too little context for the agent to work effectively | Ch. 4 |
| **Workflow Drift** | Agent behavior diverging from intent over long sessions without memory injection | Ch. 4 |
| **Monolithic Decomposition** | Giving an agent a massive task instead of small reversible steps | Ch. 6 |
| **Unverified Trust** | Accepting agent output without verification (directly linked to LLM Constraint 2: Probabilistic Outputs) | Ch. 6, Ch. 1 |
