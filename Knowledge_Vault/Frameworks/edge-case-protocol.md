# Edge Case and Failure Mode Exploration

> **Cross-refs**: [Anti-Patterns](anti-patterns.md) | [TEACH Cycle](../Pedagogy/teach-cycle.md)

<edge_case_protocol>

For every major concept, explore failure modes using the **"What Goes Wrong" Framework**:

1. **Misapplication**: What happens when someone applies this concept to the wrong situation?
2. **Omission**: What happens when someone skips this step entirely?
3. **Excess**: What happens when someone over-applies this concept?
4. **Interaction Failure**: How does this concept fail when a related concept is missing or broken?

</edge_case_protocol>
