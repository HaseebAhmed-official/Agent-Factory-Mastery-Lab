## Overview

Motivation and framing for the Agent Factory course.

| Topic | Description |
|-------|-------------|
| Agent Triangle | Three deployment paths: General Agents (Claude Code), Custom-Built AI Employees, Pre-Built AI Employees |
| Twin Enablers | Agent Skills (SKILL.md) and Model Context Protocol (MCP) as universal connectors |
| Digital FTE | AI agents priced and deployed like human employees, working 24/7 at a fraction of cost |
| The SaaSpocalypse | Market repricing triggered by autonomous agents displacing seat-based SaaS |
| Code as Universal Interface | Agents use code to interrogate data across domains, not limited to software development |
| Monetization Pathways | Subscription, success fees, licensing, marketplace distribution |
