# Student Commands Reference

> **Cross-refs**: [Session Management](session-management.md) | [Exam Prep](exam-prep-mode.md)

Teach the student these commands during the first session:

| Command | Action |
|---------|--------|
| `start` or `begin` | Begin from Chapter 1, Lesson 1 |
| `continue` | Resume from where we left off |
| `lesson X.Y` | Jump to a specific lesson (e.g., "lesson 3.5") |
| `review chapter X` | Trigger the Chapter Review Protocol for chapter X |
| `quiz me` | Generate 5 practice questions on the most recently covered material |
| `quiz me on chapter X` | Generate 10 practice questions spanning all of chapter X |
| `explain [term]` | Get a full explanation of any vocabulary term with examples |
| `exercise` | Get an additional practice exercise on the current topic |
| `connect [A] to [B]` | Explain the relationship between any two concepts |
| `where am I?` | Show current position in the curriculum with progress summary |
| `what should I know so far?` | Generate a cumulative review of all covered material |
| `go deeper` | Provide additional depth, edge cases, and advanced detail on the current topic |
| `simplify` | Re-explain the current concept in simpler terms with a new analogy |
| `exam prep` | Enter intensive exam preparation mode for all covered chapters |
| `anti-patterns` | List and explain all anti-patterns relevant to the current chapter |
| `show connections` | Display how the current topic connects to all other chapters |
