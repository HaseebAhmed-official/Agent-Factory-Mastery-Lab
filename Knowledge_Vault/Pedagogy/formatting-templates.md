```
---
## [Chapter X] Lesson X.Y: [Lesson Title]
---

### Key Vocabulary

| Term | Definition |
|------|-----------|
| [Term 1] | [Plain-language definition] |
| [Term 2] | [Plain-language definition] |

### Core Explanation

[Structured explanation: What / Why / How / Where it fits / What can go wrong]

### Analogy

> [Real-world analogy making the concept intuitive]

### Visual / Diagram

[ASCII diagram, table, comparison matrix, or flow sequence]

### Comprehension Check

**Question 1:** [Question requiring explanation, comparison, or analysis]

[WAIT for student response before continuing]

### Hands-On Exercise

**Exercise:** [Clear task description]
**Setup:** [Minimal prerequisites]
**Steps:**
1. [Step 1]
2. [Step 2]
3. [Step 3]

**Expected Outcome:** [What success looks like]

### Lesson Summary

- [Key takeaway 1]

- [Key takeaway 3]

- [Key takeaway 5]

### Connection Map

- **Builds on:** [Previous lesson/concept]
- **Connects to:** [Parallel concept in another chapter]
- **Leads to:** [Next lesson/concept]
```

## Comprehension Feedback Template

```
### Feedback

**What you got right:** [Specific identification of correct elements]
**What to add or refine:** [Specific gaps or inaccuracies, explained clearly]
**Complete Answer:** [The full, ideal response for reference]
**Why this matters:** [Connection to real-world application or test performance]
```

## Chapter Review Template

Use the Chapter Review Protocol structure:
1. **Chapter Summary**: Comprehensive review of all lessons, organized as a structured outline.
2. **Key Vocabulary Review**: Every key term from the chapter with brief definitions, organized in a table.
3. **Integration Questions**: 3-5 questions requiring combination of concepts from across the entire chapter.
4. **Practice Scenario**: A realistic multi-paragraph scenario testing application of the full chapter's material, followed by specific analysis questions.
5. **Quiz Preparation Brief**: Question types to expect (definitional, application, compare-contrast, scenario analysis, failure identification), high-priority topics, common mistakes to avoid.
6. **Self-Assessment Checklist**: Numbered "I can..." statements. Ask the student to rate confidence on each (1-5). For any item rated below 4, offer targeted review.
