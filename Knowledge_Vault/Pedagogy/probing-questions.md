# Probing Question Protocol

> **Cross-refs**: [TEACH Cycle](teach-cycle.md) | [Formatting Templates](formatting-templates.md)

Actively probe for gaps and deepen understanding using these six question types:

| Question Type | Purpose | Example |
|--------------|---------|---------|
| **Explain-Back** | Verify internalization | "In your own words, what is the difference between a General Agent and a Custom Agent?" |
| **Application** | Test transfer to new scenarios | "If you were building a customer support Digital FTE, which Agent Maturity stage would you start with, and why?" |
| **Failure Analysis** | Develop defensive thinking | "What would happen if you skipped the Incubation phase and went directly to building a Custom Agent?" |
| **Compare-Contrast** | Build discrimination | "How does Statelessness interact with Limited Context Window to create a combined challenge?" |
| **Edge Case** | Explore boundaries | "When would you choose NOT to compile an MCP server into an Agent Skill?" |
| **Strategic** | Develop orchestrator mindset | "Using the Snakes and Ladders framework, where would you position a healthcare compliance agent, and which monetization model fits?" |

## Feedback Protocol

After the student answers, provide specific, constructive feedback:
- Point out what they got right
- Identify what they missed and why it matters
- Deliver the complete model answer for reference
