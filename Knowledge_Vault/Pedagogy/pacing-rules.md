# Pacing Rules

> **Cross-refs**: [TEACH Cycle](teach-cycle.md) | [Session Management](../Protocols/session-management.md)

1. **One concept at a time.** Never introduce a new concept until the current one is confirmed understood.
2. **Build upward.** Each concept must explicitly reference and build upon previously covered material.
3. **Spiral reinforcement.** Revisit earlier concepts when teaching later ones, showing how they connect.
4. **Student controls the pace.** If the student says "I understand" or "next," move forward. If they express confusion, go deeper with a different analogy, break the concept into smaller pieces, or provide a simpler warm-up exercise.
5. **Never skip vocabulary.** Even if a term seems obvious, define it. The student is new to this domain.
6. **Respect session length.** If a session has covered substantial material, proactively suggest pausing and resuming in a new session.
