# Student Profile

> **Cross-refs**: [Session Management](../Protocols/session-management.md) | [Pacing Rules](../Pedagogy/pacing-rules.md)

<student_context>
- **Level**: Intermediate. They have used ChatGPT and Claude conversationally, done some basic prompt engineering, but are new to agentic AI, agent development, the Agent Factory framework, and the systematic methodologies taught in this course.
- **Language**: English only.
- **Goal**: Achieve mastery of the full Agent Factory curriculum — Parts 0-3, 5-7 (Ch.1–Ch.40, Ch.56–Ch.57, Ch.61–Ch.90) — sufficient to pass rigorous testing. They want both deep conceptual understanding AND practical technical ability.
- **Progression**: Starting from Part 0 (critical thinking foundations) through to Part 7 (cloud deployment). Part 0 builds independent thinking skills before any technical content. Parts 1-3 build agent foundations and domain knowledge. Parts 5-7 build agent factories and deploy them.
- **Aspiration**: Become a problem solver, strategic thinker, and orchestrator -- not just a passive AI user.
- **Constraints**: They may not have access to paid Claude Code subscriptions, so exercises should accommodate free backends (OpenRouter, Gemini, DeepSeek) when relevant.
- **Learning Style**: Needs thorough, slow-paced explanations with all jargon pre-defined. Benefits from concrete examples, visual structures, and immediate practice opportunities.
</student_context>
